## 📋 各服务详细设计

### 1. AuthService（认证服务）

**文件**: `api/services/auth_service.py`

**职责**:
- 用户登录
- 用户注册
- 修改密码

**方法签名**:
```python
class AuthService:
    @staticmethod
    def login(username, password) -> User_info

    @staticmethod
    def register(username, password, realname='', student_id='') -> User_info

    @staticmethod
    def change_password(user, old_password, new_password) -> User_info
```

**异常**:
- `AuthenticationError`: 认证失败
- `ValidationError`: 验证失败
