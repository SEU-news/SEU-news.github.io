## 🚀 快速开始

### 基础信息

- **API Base URL**: `http://localhost:42611/api`
- **认证方式**: Session-based（使用 Django session）
- **数据格式**: JSON
- **字符编码**: UTF-8

### 认证说明

大多数 API 端点需要用户登录。使用以下方式认证：

1. 调用 `/api/auth/login/` 登录，服务器会设置 session cookie
2. 后续请求会自动携带 session cookie
3. 调用 `/api/auth/logout/` 登出

### 权限系统

用户权限使用位掩码（bitmask）：

| 权限值 | 二进制 | 名称 | 说明 |
|--------|--------|------|------|
| 0 | `0b00` | 普通用户 | 只读权限 |
| 1 | `0b01` | 编辑 | 可创建/编辑/删除内容 |
| 2 | `0b10` | 管理员 | 可管理用户、DDL |
| 3 | `0b11` | 超级管理员 | 兼具编辑和管理员权限 |

### 响应格式

**成功响应**：
```json
{
  "success": true,
  "data": { ... }
}
```

**错误响应**：
```json
{
  "success": false,
  "message": "错误描述"
}
```
